# 02 - Group Problem Statement

Phần này sẽ được cập nhật sau khi nhóm thảo luận và chốt một candidate problem.

Cần có trong bản nộp cuối:

- Nhật ký hội tụ từ các candidate problems về một bài.
- Kiểm chứng nhanh / validation.
- Research giải pháp đã có.
- Workflow hiện tại và workflow sau tối ưu.
- Problem Statement v0/v1 với metric và boundary rõ.
- So sánh No AI / Rule / Workflow / Agent.
- Quyết định cuối: Go / Not Yet / No-Go và lý do.
